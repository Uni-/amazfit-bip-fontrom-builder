# amazfit-bip-fontrom-builder

HTML5 FontROM builder for Xiaomi Amazfit Bip.

Runs on premise with Node.JS, or served as serverless web application.

## Dependencies

* webpack
* iconv-lite
* axios

## How to build

* `npm i`
* `npm run build`
